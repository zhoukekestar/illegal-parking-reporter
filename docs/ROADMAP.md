## P0 验收标准

1. 设置页 → YOLOv8 显示绿色「已就绪」
2. 图片检测页选 JPG → 点「运行检测」
3. 表格列出 car/truck/bus 等结果, 含置信度和边界框
4. M1 推理耗时 < 200 ms

## 阶段切换原则

每阶段达成验收才进下一阶段, 不带技术债往后走。
完成阶段后 `git tag pX-done` 打标记, 方便回滚。
